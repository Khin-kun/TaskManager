### 👉 Install with github site
1. Download the zip
2. Extract folders to a folder of your choice
3. Open folder in the terminal
   
   ```bash
      cd "Folder Path"
    ```
4. Launch the fle in the terminal
   
    ```bash 
      python main.py
     ```

### 👉 Install with git

1. Open the terminal
    ```bash
    git clone https://github.com/Khin-kun/TaskManager.git
    ```

3. Open folder
    ```bash
   cd Folder Path
     ```

5. Launch the file:
    ```bash
   python main.py
    ```
